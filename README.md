# URL Shortener - Shrink

A modern URL shortener built with **Next.js**, **Tailwind CSS**, and **MongoDB**.  
Easily shorten long URLs, manage them, and share with others.

---

## Features

- Shorten long URLs with a single click  
- View shortened URLs  
- Responsive design with Tailwind CSS  
- API routes powered by Next.js and MongoDB  
- Easy to deploy on Vercel
